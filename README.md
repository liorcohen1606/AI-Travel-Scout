# AI Travel Scout 

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://ai-travel-scoutgit-b2th6jbnmeaqmjg6mdjqun.streamlit.app/)

**[Click here to try the live app!](https://ai-travel-scoutgit-b2th6jbnmeaqmjg6mdjqun.streamlit.app/)**

An intelligent travel research agent built to generate dynamic, personalized itineraries. Instead of relying on static LLM knowledge, this agent actively searches the web to find up-to-date hidden gems, local tips, and tailored hotel recommendations based on the user's specific "travel vibe."

## Tech Stack
* **Orchestration:** LangGraph (ReAct Agent architecture)
* **LLM:** Google Gemini 2.5 Flash
* **Tools:** Tavily Search API (Real-time web browsing) & Wikipedia REST API (Dynamic images)
* **Frontend:** Streamlit
